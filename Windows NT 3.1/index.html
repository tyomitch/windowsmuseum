На главную

Windows NT 3.1

Windows NT появилась, по большому счёту, случайно. Microsoft планировала сделать новую версию OS/2 для процессора "нового поколения" N10. Проект получил название "NT OS" именно в честь этого нового процессора – N-Ten. В кратчайшие сроки команда разработчиков написала с нуля новую, переносимую ОС с POSIX-подсистемой для тестирования. Позже была добавлена OS/2-подсистема, а затем, когда Microsoft поссорилась с IBM и прекратила работу над OS/2 – Windows-подсистема, с таким же интерфейсом, как у популярной тогда Windows 3.1. Так NT OS стала Windows NT 3.1; расшифровку "New Technology" придумали маркетологи, когда проект был уже близок к завершению.

Windows NT 3.1

Windows NT 3.1 – гибрид Windows и OS/2 на совершенно новом UNIX-подобном ядре. Также это единственная версия Windows, достаточно полно поддерживающая OS/2-приложения и файловую систему HPFS.


Установка: приглашение

Windows NT нельзя было устанавливать из-под DOS, Windows и подобных операционнных систем; для запуска программы установки нужно было загрузиться с дистрибутивного CD-ROM-а или специальной поставлявшейся с ним загрузочной дискеты. При сетевой установке сначала все файлы дистрибутива копировались на локальный диск, потом создавалась такая загрузочная дискета, компьютер перезагружался, и только тогда уже запускалась программа установки.

Диспетчер задач

новая возможность в Диспетчере задач Windows NT: он позволяет не только переключаться между запущенными программами и завершать их, но и запускать новые

Ctrl-Alt-Del

Комбинация Ctrl-Alt-Del в Windows NT играла фундаментальную роль – даже для входа в систему требовалось нажать на "заветные три кнопки". Из недокументированной и малоизвестной функции IBM PC комбинация Ctrl-Alt-Del превратилась в наиболее часто используемую.
В Windows NT она вызывает "экран безопасности", который позволяет заблокировать компьютер, сменить пароль, выйти из системы и т.д.

Windows NT 3.1 Beta

За полгода до релиза вышла бета-версия Windows NT. Отличия её от окончательной версии минимальны; они в основном в том, что поддерживалось гораздо меньше устройств, и не входило некоторых стандартных программ, таких как FTP-сервер. Однако даже и Windows NT 3.1 SP3 поддерживала слишком мало устройств, чтобы стать сколько-нибудь распространённой. Например, не поддерживались IDE CD-ROM приводы.
