Windows 2.x

Версия Windows 2.x впервые сделала Windows заметной как операционную систему. Первые версии Word for Windows требовали именно Windows 2.10, но было также и множество сторонних программ для этих версий Windows. До мирового лидерства было ещё, конечно, очень далеко.
Windows 2.x распространялась в виде рантайм-версии с некоторыми прикладными программами. Так распространялся, например Excel 2.1; сторонние компании также могли лицензировать рантайм-версию для распространения со своими программами. Вроде бы существовала и рантайм-версия Windows 1.x; более старшие версии Windows в виде рантайм-версий уже не распространялись.
Отличительная особенность Windows 2.x – её можно запустить прямо из-под Windows 9x, правда, без поддержки мыши.

Windows 2.03

Windows 2.03 отличался от Windows 1.x усовершенствованным интерфейсом, разрешением перекрытия окон, поддержкой мыши PS/2.

Буфер обмена

На этом скриншоте видны два изменения в интерфейсе Windows 2.x: во-первых, добавились горячие кнопки для выполнения операций над окнами (уже в Windows 3.0 их убрали, так что это "отличительная особенность" Windows 2.x); во-вторых, пункт "About" у стандартных программ перенесён из системного меню окна, где он был в Windows 1.x, в первое слева меню (обычно "File"). У Буфера обмена нет других меню, кроме системного, и поэтому у него пункт "About" остался на прежнем месте.

Элементы управления окном

В Windows 2.x интерфейс управления окнами был капитально пересмотрен: теперь в правом верхнем углу окна имелось две кнопки – "свернуть" и "развернуть/восстановить"; кроме того, размер окна можно менять, потянув мышью за его толстую рамку.
Ещё одно нововведение Windows 2.x в этой области – окна с тонкой рамкой, у которых нельзя менять размер (например, таковы Панель управления и Калькулятор) – у таких окон в правом верхнем углу только одна кнопка – "свернуть".

Windows 2.1

Windows 2.1 и 2.11 существовали в двух ипостасях каждый: Windows/286 и Windows/386; таким образом, в продаже были пять версий Windows 2.x. Windows 2.1x использовали новые возможности по управлению памятью современных себе процессоров. Соответственно, Windows/286 мог (потенциально) адресовать 16Мб память, а Windows/386 – 4Гб; однако различия, по существу, только в этом и состояли.
