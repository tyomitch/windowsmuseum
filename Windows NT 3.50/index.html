Windows NT 3.5x

Windows NT 3.50 отличалась от предыдущей версии, в первую очередь, более широкой поддержкой устройств. Например, это первая версия Windows NT, которая поддерживает IDE CD-ROM-приводы и виртуальный сетевой адаптер VMWare.
Через полгода после Windows NT 3.50 вышла усовершенствованная версия – Windows NT 3.51, бывшая последней ОС семейства Windows с интерфейсом в стиле "Без кнопки Пуск". Её отличия от Windows NT 3.50 ещё менее заметны, чем той – от Windows NT 3.1.

Windows NT 3.50

Мне не удалось заставить заработать окончательную версию Windows NT 3.50, поэтому здесь представлены скриншоты предварительной версии – Release Candidate. В ней нет некоторых из возможностей, появившихся в окончательной версии – в том числе, заявленной совместимости с IDE CD-ROM-приводами и сетевой картой AMD PCNET.
Эта предварительная версия распространялась, по-видимому, только в серверном варианте; по крайней мере, пока это – единственная серверная версия Windows NT в моей коллекции.


Установка: приглашение

Программу установки Windows NT 3.5x можно было запустить тремя разными способами: загрузившись с дистрибутивного CD-ROM-а, из-под MS-DOS, либо из-под Windows NT. Этот третий способ, впервые появившийся в Windows NT 3.50, предназначался для обновления предыдущих версий Windows NT. При нём, как и при установке из-под MS-DOS, необходимые файлы копировались во временную папку, создавались загрузочные дискеты, и после перезагрузки начиналась собственно установка.

Установка: выбор каталога

В отличие от всех остальных версий Windows NT, версии 3.5x предлагали в качестве папки для установки \WINNT35. По видимому, это путало карты всем программам, предполагавшим что Windows NT устанавливается в папку \WINNT, и начиная с Windows NT 4 её вновь сделали папкой по умолчанию.
при обновлении старой версии Windows NT программа установки могла назвать её версию

Windows NT 3.51 Shell Update

Вскоре после выхода Windows NT 3.51 для него вышел "Shell Update" в виде обновлений системных файлов, т.е. без программы устновки. Он имел статус бета-версии Windows NT 4 и поэтому не был в продаже, а распространялся группам бета-тестеров. Он изменял интерфейс на Windows 95-подобный, при этом оставляя всё "нутро" системы старым. Обновление тоже было недостаточно гладким; например, появляющаяся Корзина была общей для всех пользователей (в окончательной версии Windows NT 4 у каждого пользователя была уже своя корзина).

Элементы управления окном

Элементы управления окном – то, что наиболее заметно изменилсь в Windows NT 3.51 SU. Вместо "минусика", соответствующего системному меню окна, отображается его значок; справа от кнопок "свернуть" и "развернуть/восстановить" (из стрелок превратившихся в пиктограммы) появилась кнопка "закрыть" в виде крестика. Ещё одно изменение интерфейса в том, что пункты меню по умолчанию (на скриншоте – "Close") выделены жирным шрифтом.
