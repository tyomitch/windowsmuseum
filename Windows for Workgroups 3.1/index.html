Windows for Workgroups

В продаже были две версии Windows для рабочих групп – WfW 3.1 и WfW 3.11. Это подсемейство Windows было рассчитано для маломощных рабочих станций в корпоративных сетях.

Windows for Workgroups 3.1

Отличие обычной Windows 3.1 от Windows 3.1 for Workgroups состояло в поддержке сетей распространённых тогда технологий; соответственно, в WfW были дополнительно включены сетевые программы, такие как почтовый клиент MS Mail и служба сообщений WinPopup.

Windows for Workgroups 3.11

Windows for Workgroups 3.11 был венцом ветви Windows 3.1x. В отличие от WfW 3.1, он мог работать и без сети. Кроме того, WfW 3.11 поддерживал только расширенный режим. Наконец, WfW 3.11 был первой версией Windows, получившей в России широкое распространение.

Командная строка

Win32s

Вместе с распространением новой 32-битной ОС Windows NT Microsoft выпустила подсистему, позволявшую запускать некоторые из приложений Win32 в расширенном режиме Windows 3.1x. Система эта была очень ограничена в возможностях, и запускались под ней в основном только те приложения, которые специально для неё были написаны. Однако это был очень важный переходный этап, позволявший разработчикам писать и тестировать Win32-приложения под Windows 3.1x, а пользователям – использовать новейшие Win32-приложения без необходимости переходить на новую ОС.
