Windows 3.0

Windows 3.0 представляла собой три версии в одной: она могла работать в реальном (8086), стандартном (286) и расширенном (386) режимах. Интерфейс был вновь в корне пересмотрен; добавилась псевдо-трёхмерность некоторых элементов.

Windows 3.0

В продаже были две версии – Windows 3.00 и Windows 3.00a; последняя отличается только исправленными багами.


Установка: приглашение

установка в Windows 3.0 стала более дружественной, чем в предыдущих версиях; вместо предъявления пользователю требований, что он должен знать, та предлагает ему контекстную справку на каждом этапе установки

Установка: выбор каталога

установка Windows 3.00 впервые различает ситуации новой установки Windows...
...и обновления предыдущей версии
Windows 3.00a дополнительно различает обновление Windows 2.x...
...и Windows 3.00

Установка: копирование файлов

установка состоит из двух этапов, текстового (ниже) и графического
Windows 3.0 предлагает сама найти и установить DOS-приложения, уже имеющиеся на дисках

Установка: готово

поскольку Windows 3.0 может работать в реальном режиме, то её можно запустить сразу же после установки, без перезагрузки
обновление Windows 3.00 до Windows 3.00a, напротив, состоит только из текстового этапа, и требует перезагрузку

Язык и стандарты

Свойства системы

хотя уже Windows/386 2.1 мог использовать преимущества 386-го процессора – так называемый "расширенный режим" – соответствующие ему настройки, позже выросшие в "свойства системы", появились только в Windows 3.0

О системе

реальный режим: доступно 350Кб памяти
стандартный режим: доступно 13Мб памяти
расширенный режим: доступно 34Мб памяти
На этой виртуальной машине установлено только 16Мб памяти; вся остальная память обеспечивается файлом подкачки.
кроме окна "О системе" Windows 3.00a можно отличить от Windows 3.00 только с помощью winver в командной строке

Файловый менеджер

Редактор PIF

редактор PIF теперь позволяет создавать два типа PIF-файлов: для стандартного и расширенного режимов Windows
набор дополнительных параметров PIF-файлов расширенного режима впечатляет
PIF-файлы стандартного режима не имеют дополнительных параметров: все доступные параметры задаются в одном окне

Редактор настроек

В Windows 3.0 впервые включена утилита для редактирования файлов настройки. Автоматически открываются 4 файла – AUTOEXEC.BAT, CONFIG.SYS, SYSTEM.INI, WIN.INI

Командная строка

В состав Windows теперь входит утилита winver, которая в командной строке выводит номер текущей версии, а при непосредственном запуске – окно с информацией о версии.
Кроме того, окно командной строки теперь закрывается автоматически.

Игра

это последнее появление реверси в Windows; в последующих версиях его почему-то заменили на сапёр
зато это первая версия Windows, в которую входят карточные игры; в будущих версиях они будут явно доминировать над логическими играми вроде реверси и сапёра

Справка

также это первая версия Windows со встроенной справкой

Элементы управления окном

элементы управления окном в Windows 3.0 практически не изменились по сравнению с Windows 2.x; интереснее отметить другие особенности окна командной строки – что теперь оно, даже развёрнутое, занимает не весь экран, и что "минусик", соответствующий системному меню, теперь перекрашивается при нажатии

Диспетчер задач

одно из новшеств Windows 3.0 – Диспетчер задач, вызываемый комбинацией клавиш Ctrl-Esc; начиная с Chicago, эта комбинация была переназначена для вызова главного меню
