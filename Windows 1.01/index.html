Windows 1.x

Изначально Windows задумывалась как графический движок для прикладных программ; именно в этом качестве мир впервые услышал о ней в 1983-1984 г. Затем Microsoft справедливо решила, что одного движка недостаточно, чтобы покорить рынок, и стала продавать его вместе с пакетом "демонстрационных" программ. Windows 1.00, выпущенная в 1985 г., содержала какие-то ошибки, и была изъята из продажи за пару недель. Первой реально продаваемой версией Windows была 1.01

Элементы управления окном

Обычно в полоске снизу экрана показываются свёрнутые приложения, а вся остальная часть экрана занята мозаикой из активных приложений – окна в Windows 1.x не могли перекрываться.
Прямоугольнички в правых углах окна – то самое место, за которое их размер можно менять мышью.
Окно может быть развёрнуто на весь экран; только тогда оно закрывает полоску свёрнутых приложений внизу.
Когда одновременно развёрнуто несколько окон, становятся доступны команды Move и Size для управления мозаикой из окон.
Команда Move позволяла "бросить" одно из окон на границу другого, и тогда брошенное окно получало половину места прежнего окна с той стороны, куда оно было брошено.
Команда Size действовала в два этапа: сначала выбиралось, какую из границ перемещать, затем – на сколько. Таким образом, изменение размера всегда затрагивало два или более окна.

Windows 1.04

Windows 1.04 – последняя версия Windows 1.x; таким образом, в продаже были четыре её различные версии. Из них Windows 1.03 добавила несколько усовершенствований, таких как наличие языковых установок и возможность выделения в окне командной строки без мыши; Windows 1.04 по сравнению с ней содержала в основном исправления багов.

Логотип

Большой логотип с обложки папки с конференции Windows-разработчиков. Если приглядеться, то на фоне можно разглядеть "блиббеты" ("зубастые" буквы O), бывшие в то время логотипом Microsoft.
Маленький логотип с программки конференции. Не в точности совпадает с большим (в центре окна – эффект свечения).
Слоган Windows 1.0: "Open the Windows of the future". Сама конференция проходила под девизом "The Future Starts Here."
первый в истории знак "Designed for Windows": подпись гласит "AN APPLICATION DESIGNED FOR MICROSOFT WINDOWS"
